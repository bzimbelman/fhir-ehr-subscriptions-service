{{- if and .Values.autoscaling.enabled (hasKey .Values "replicaCount") }}

WARNING (OP #124):
  Both `replicaCount` ({{ .Values.replicaCount }}) and
  `autoscaling.enabled` are set. The Deployment template intentionally
  omits .spec.replicas when autoscaling is on, so `replicaCount` has NO
  effect at install time — the HPA's `minReplicas`
  ({{ .Values.autoscaling.minReplicas }}) controls the floor instead.
  Either set autoscaling.minReplicas to your desired floor and remove
  the replicaCount override, or set autoscaling.enabled=false to use a
  static replicaCount.
{{- end }}
